# field operator

physics sims for the accelerate channel

## features

- **coulomb fields** - electrostatics with particle spawning, left click for positive charges, right click for negative
- **rf waveguide** - em wave propagation, place transmitters and receivers, watch waves travel
- **oscilloscope** - lissajous curves with keyboard controls (Q/W/A/S/Z/X/C/V), press M for microphone input
- **tokamak** - fusion plasma confinement, magnetic field controls with keyboard (1-6), press R to reset

## controls

each mode has its own sliders and keyboard shortcuts, check the instruction popup when you enter a mode

## setup

just open index.html or host it somewhere

live demo: https://xapqrt.github.io/phy-acc/
